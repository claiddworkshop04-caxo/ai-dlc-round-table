# INT-000 UNIT-003: 貸出状況ダッシュボード

## 目的
貸出中の備品一覧と期限超過ハイライト、全貸出履歴の表示機能を実装する。
管理者が現在の貸出状況を素早く把握できるトップページとして提供する。

## 担当 User Stories
- US-006: 貸出状況を一覧で確認する
- US-007: 返却期限超過の備品を把握する

## 境界 (Boundaries)
- IN: / ダッシュボード（貸出中一覧・期限超過ハイライト）、/loans 貸出履歴一覧
- OUT: 貸出・返却操作、マスタ管理

## 依存関係
- 依存: UNIT-001（items/loansテーブル）、UNIT-002（貸出レコードが存在すること）

| Bolt ID | タイトル |
|---------|----------|

# INT-000 UNIT-002: QRスキャン・貸出返却

## 目的
QRコードをスキャンして備品の貸出・返却操作を行う機能を実装する。
メンバーリストからの借用者選択とフリー入力の両方に対応する。

## 担当 User Stories
- US-004: QRスキャンで備品を借りる
- US-005: QRスキャンで備品を返却する

## 境界 (Boundaries)
- IN: QRスキャン画面、貸出フォーム（メンバー選択/フリー入力）、返却確認フォーム、loans CRUD
- OUT: 備品・メンバーのマスタ管理、ダッシュボード表示

## 依存関係
- 依存: UNIT-001 完了（items/members/loansテーブル、備品マスタ・メンバーマスタ確立）
- 新規依存: `html5-qrcode` npm パッケージ

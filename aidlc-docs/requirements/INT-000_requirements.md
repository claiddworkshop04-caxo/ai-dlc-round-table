# INT-000 機能要件 (Requirements)

## FR-001: 備品マスタ管理
- FR-001-1: 備品を登録できる（備品名、説明、標準返却日数）
- FR-001-2: 備品を編集できる
- FR-001-3: 備品を削除できる（貸出中の場合は削除不可）
- FR-001-4: 備品一覧を表示できる

## FR-002: QRコード管理
- FR-002-1: 備品ごとにQRコードを生成できる（備品IDをエンコード）
- FR-002-2: QRコードをPNG形式でダウンロード、または画面表示できる

## FR-003: メンバーマスタ管理
- FR-003-1: メンバーを登録できる（氏名）
- FR-003-2: メンバーを編集できる
- FR-003-3: メンバーを削除できる
- FR-003-4: メンバー一覧を表示できる

## FR-004: 貸出操作
- FR-004-1: QRコードをスキャンすると対象備品の貸出画面が開く
- FR-004-2: 借用者をメンバーリストから選択できる
- FR-004-3: メンバーリストにない場合はフリーテキストで借用者名を入力できる
- FR-004-4: 貸出日は操作日（当日）を自動セット
- FR-004-5: 返却期限は「貸出日 + 備品マスタの標準返却日数」で自動計算する
- FR-004-6: 貸出中の備品にはスキャン時に「返却」ボタンを表示する（再貸出不可）

## FR-005: 返却操作
- FR-005-1: QRコードをスキャンすると対象備品の返却確認画面が開く
- FR-005-2: 返却確認を行うと貸出レコードの返却日時を更新し、ステータスを「返却済」にする

## FR-006: 貸出状況一覧
- FR-006-1: 現在貸出中の備品を一覧で表示する（備品名・借用者・貸出日・返却期限）
- FR-006-2: 返却期限超過の行を視覚的に強調表示する（赤色など）
- FR-006-3: 返却済みを含む貸出履歴を一覧で表示できる（オプション表示）

## FR-007: QRスキャン画面
- FR-007-1: Webカメラを使ってQRコードをスキャンできる
- FR-007-2: スキャン成功時に対象備品の貸出/返却画面へ遷移する

## データ制約
- 備品の「標準返却日数」は 1 以上の整数
- 同一備品の同時貸出は1件のみ（貸出中は再貸出不可）
- 借用者名（フリー入力）は最大100文字

# INT-000 Intent: 備品貸出管理システム

## 目的
オフィス内の備品（プロジェクター・ノートPC・延長コードなど）の貸出・返却をQRコードで管理し、「どの備品が誰に貸出中か」を即座に把握できるWebアプリを構築する。

## 成功基準
1. 備品にQRコードを貼付し、スキャンするだけで貸出・返却の操作が完了する
2. 管理者が現在の貸出状況（誰がどの備品を借りているか）を一覧で確認できる
3. 返却期限を自動計算し、期限超過の備品を視認できる
4. 事前登録メンバーのリストから借用者を選択でき、未登録者はフリー入力で対応できる

## スコープ

### 対象
- 備品マスタ管理（登録・編集・削除、標準返却日数の設定）
- メンバーマスタ管理（登録・編集・削除）
- QRコード生成（備品ごと）
- QRコードスキャンによる貸出・返却操作
- 貸出状況一覧（現在の貸出中備品の一覧）
- 返却期限の自動計算（貸出日 + 標準返却日数）
- 返却期限超過の視覚的表示

### 対象外
- 認証・ログイン機能
- メール/Slack通知
- 多拠点対応
- モバイルアプリ（PWAも対象外）
- レポート・統計機能

## 技術スタック
- フロントエンド/バックエンド: Next.js 16 (App Router) + Server Actions
- ORM: Drizzle ORM
- DB: Neon PostgreSQL
- UI: shadcn/ui + Tailwind CSS
- QR生成: qrcode ライブラリ
- QRスキャン: html5-qrcode または jsQR
- デプロイ: Netlify

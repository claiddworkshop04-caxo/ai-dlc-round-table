# INT-000 US-006: 貸出状況を一覧で確認する

## As a
管理者

## I want
現在の貸出状況を一覧で確認したい

## So that
どの備品が誰に貸出中かを即座に把握できる

## Acceptance Criteria
- [ ] 貸出中の備品の一覧が表示される（備品名・借用者・貸出日・返却期限）
- [ ] 全履歴（返却済み含む）を表示する切り替えができる

# INT-000 US-007: 返却期限超過の備品を把握する

## As a
管理者

## I want
返却期限を過ぎた備品を一目で確認したい

## So that
催促が必要な貸出を素早く特定できる

## Acceptance Criteria
- [ ] 返却期限超過の行が赤色などで強調表示される
- [ ] 期限超過かどうかは現在日付と返却期限を比較して判定する

# INT-000 US-001: 備品マスタを管理する

## As a
管理者

## I want
備品の登録・編集・削除を行いたい

## So that
管理対象の備品情報（備品名・説明・標準返却日数）を常に正確に保ちたい

## Acceptance Criteria
- [ ] 備品名・説明・標準返却日数を入力して備品を登録できる
- [ ] 登録した備品が一覧に表示される
- [ ] 備品の情報を編集できる
- [ ] 貸出中でない備品を削除できる
- [ ] 標準返却日数は1以上の整数のみ登録できる

# INT-000 US-002: メンバーマスタを管理する

## As a
管理者

## I want
メンバー（氏名）の登録・編集・削除を行いたい

## So that
貸出時の借用者選択で正確なメンバーリストを提供できる

## Acceptance Criteria
- [ ] 氏名を入力してメンバーを登録できる
- [ ] 登録したメンバーが一覧に表示される
- [ ] メンバーの氏名を編集できる
- [ ] メンバーを削除できる

---

# INT-000 US-003: 備品のQRコードを生成・表示する

## As a
管理者

## I want
備品ごとのQRコードを生成して画面表示・ダウンロードしたい

## So that
備品に貼付するQRコードを簡単に準備できる

## Acceptance Criteria
- [ ] 備品一覧からQRコード生成ボタンを押すとQRコードが表示される
- [ ] QRコードには `/items/[id]/loan` のURLがエンコードされている
- [ ] QRコードをPNG形式でダウンロードできる

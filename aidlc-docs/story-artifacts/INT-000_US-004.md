# INT-000 US-004: QRスキャンで備品を借りる

## As a
利用者

## I want
備品に貼られたQRコードをスキャンして貸出操作をしたい

## So that
手入力なしで素早く備品を借りることができる

## Acceptance Criteria
- [ ] QRスキャン画面でカメラが起動する
- [ ] QRコードをスキャンすると対象備品の貸出画面に遷移する
- [ ] 借用者をメンバーリストから選択できる
- [ ] メンバーリストにない場合はフリーテキストで借用者名を入力できる
- [ ] 貸出日は当日が自動セットされる
- [ ] 返却期限は「貸出日 + 備品マスタの標準返却日数」で自動計算・表示される
- [ ] 貸出ボタンを押すと貸出レコードが作成される
- [ ] 既に貸出中の備品はスキャン時に「貸出中」と表示され、再貸出できない

# INT-000 US-005: QRスキャンで備品を返却する

## As a
利用者

## I want
備品のQRコードをスキャンして返却操作をしたい

## So that
返却を記録し、次の借用者が利用できる状態にできる

## Acceptance Criteria
- [ ] 貸出中の備品のQRコードをスキャンすると返却確認画面が表示される
- [ ] 返却確認ボタンを押すと返却日時が記録され、ステータスが「返却済」になる
- [ ] 返却後、その備品は再び貸出可能になる

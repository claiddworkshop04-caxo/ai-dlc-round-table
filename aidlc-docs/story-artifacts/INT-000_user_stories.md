# INT-000 User Stories 一覧

## ペルソナ
- **管理者**: 備品・メンバーの登録や貸出状況を確認するオフィス担当者
- **利用者**: 備品を借りる・返却するオフィスメンバー

## User Stories

| ID | タイトル | ペルソナ | 優先度 | 担当Unit |
|----|----------|----------|--------|----------|
| US-001 | 備品マスタを管理する | 管理者 | Must | UNIT-001 |
| US-002 | メンバーマスタを管理する | 管理者 | Must | UNIT-001 |
| US-003 | 備品のQRコードを生成・表示する | 管理者 | Must | UNIT-001 |
| US-004 | QRスキャンで備品を借りる | 利用者 | Must | UNIT-002 |
| US-005 | QRスキャンで備品を返却する | 利用者 | Must | UNIT-002 |
| US-006 | 貸出状況を一覧で確認する | 管理者 | Must | UNIT-003 |
| US-007 | 返却期限超過の備品を把握する | 管理者 | Must | UNIT-003 |
